# Versitron MF7275-2SFB | 10/100TX-FX Single Fiber Industrial Media Converter (ST) For System Integrator

The Versitron MF7275-2SFB is a rugged 10/100TX to FX industrial media converter designed for system integrators who require reliable Ethernet-to-fiber connectivity in harsh environments. It converts copper Ethernet signals into singlemode fiber over a single strand (BiDi) using an ST connector, enabling long-distance communication for industrial, security, and transportation networks.
